Support

YouTube Activity Cleaner

This page is the support destination for the Chrome Web Store listing and for users who need troubleshooting help.

Quick Troubleshooting

  • If the popup says it cannot connect to the page, reload the Google My Activity comments page once after reloading the extension.
  • Keep the Google My Activity tab visible while the cleaner is running. Hidden tabs are throttled by Chrome.
  • The current beta is best tested with the English and Polish Google My Activity interface.
  • If the page layout changed, try the Safe profile and slower pacing first.
  • If Google temporarily rejects a delete action, the cleaner can retry automatically based on your popup settings.

Report A Bug

The preferred support channel is the GitHub issue tracker: github.com/MichalMatu/youtube-activity-cleaner/issues

When possible, include:

  • the extension version and UI language shown in the popup `About` line
  • your Chrome version
  • whether you used Fast or Safe mode
  • the popup status message or last error
  • a screenshot of the Google My Activity page if the layout changed
