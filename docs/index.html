Chrome Extension

YouTube Activity Cleaner

Clean old YouTube comments from the Google My Activity comments page with a focused, local-first MV3 extension.

View GitHub Repository Support Privacy policy

What It Does

Finds visible delete actions on the Google My Activity comments page, confirms them, waits for the page to react, and repeats.

What It Does Not Do

It does not sign you in, sync data to a server, or store your YouTube history outside your browser.

Key Features

Fast or Safe mode. Choose a more aggressive run profile or a more conservative one before starting.

Retry and backoff. The cleaner retries failed delete flows with clear status messages and failure protection.

Local settings. Pacing and retry preferences are saved locally in Chrome and reused on the next run.

No cloud backend. The extension uses browser permissions only for tabs, power keep-awake, and local storage.

How To Use

  1. Load the unpacked extension from the extension/ directory or install it from the Chrome Web Store later.
  2. Open the Google My Activity page for Your YouTube comments.
  3. Open the popup, review the settings, and click Start.
  4. Keep the Google My Activity tab visible while the cleaner is working.
  5. Use the popup to monitor progress or click Stop.

The current beta is best tested with the Google My Activity interface in English and Polish.

Current Status
